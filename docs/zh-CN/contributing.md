# 贡献指南

欢迎为 JSON Shape Explorer 贡献想法、问题反馈、文档和代码。

## 适合新贡献者的方向

- 补充测试用例
- 改进中文或英文文档
- 添加更多示例 JSON
- 改进错误提示
- 实现新的输出格式

## 本地开发

运行示例：

```bash
node src/cli.js examples/users.json
```

运行测试：

```bash
node --test
```

## 代码风格

当前项目使用原生 JavaScript 和 Node.js 标准库，尽量保持零依赖。

提交代码时请尽量遵守：

- 保持函数短小清楚
- 优先添加测试 fixture
- 避免引入不必要的依赖
- 文档和示例要能直接运行

## 提交 Issue

提交 issue 时，请尽量说明：

- 你使用的输入 JSON 示例
- 你期望看到的输出
- 当前实际输出
- 你认为问题属于 bug、功能建议还是文档改进

## 提交 Pull Request

一个好的 PR 通常包含：

- 清晰的变更说明
- 对应测试
- 必要的文档更新
- 尽量小而聚焦的改动范围

如果你不确定某个功能是否适合加入，可以先开 issue 讨论。
